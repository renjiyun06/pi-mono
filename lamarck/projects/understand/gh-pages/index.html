How it works

understand file.ts

Generates 3 targeted comprehension questions about any code file. Tests design decisions, failure modes, and runtime behavior.

understand debt

Shows which recently changed files you've never been quizzed on. Tracks cognitive debt across your codebase.

Git post-commit hook

Automatically quizzes you after commits with 20+ line changes. Opt-in via UNDERSTAND_AUTO_QUIZ=1.

MCP server

5 tools for AI coding agents. Quiz, evaluate, score, diff analysis, and debt tracking ‚Äî all via Model Context Protocol.

What it's not

Not a linter (those check syntax). Not a test framework (those check behavior). Not an explanation tool (those show you the answer).

It checks whether you understand what you shipped. That's the gap nothing else fills.
