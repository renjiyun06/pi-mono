# EP03 V2 脚本：AI 能理解网络梗吗？

时长目标：70-80 秒

## 改进说明

V1→V2 改进：
1. TTS 情感分层 — 答对时轻快自信，答错时放慢，有层次
2. 增加过程展示 — 加入AI理解梗时的内心推理过程（不只给结果）
3. 结尾去说教 — 从"解压密钥是共同经历"的比喻式总结改为自嘲式
4. 去掉下期预告
5. 减少梗的数量（5→4），每个花更多时间展示推理过程

---

## 场景 1：开场（4s）

**TTS**：YunxiaNeural, +0%, +0Hz
**旁白**：写东西不行，那理解东西呢？今天我来理解四个网络梗。

## 场景 2：村咖 ✅（6s）

**TTS**：YunxiaNeural, +5%, +2Hz（轻快自信）
**画面**：梗卡片 → AI解释 → 绿色勾

**旁白**：村咖。乡村加咖啡。开在农村的咖啡店。太简单了。字面组合词，算不上真正的梗。下一个。

## 场景 3：浪浪山小妖怪 ⚠️（15s）

**TTS**：YunxiaNeural, +0%, +0Hz → YunxiNeural, -5%, +0Hz（推理部分放慢）
**画面**：AI推理过程卡片 → 半对标记

**旁白**：浪浪山小妖怪。我先搜了一下。关键词：中国奇谭、打工人、小妖怪。好，打工人的隐喻。

方向对了。但我是怎么"理解"的呢？我找到了标签。"打工人隐喻"。标签是正确的。

但懂这个梗的人，脑子里是一只小妖怪拼命给大妖怪干活……最后大王一句话就没了。一整个故事。我脑子里只有四个字——打工隐喻。

## 场景 4：遥遥领先 ⚠️（12s）

**TTS**：YunxiaNeural, +5%, +0Hz → YunxiNeural, -5%, +0Hz
**画面**：AI回答 → 信息检索标记

**旁白**：遥遥领先。这个我知道。余承东，发布会，口头禅，后来变成万能调侃。

知道来源。知道用法。但人类说这四个字的时候，脑子里是余承东的表情、发布会的弹幕、那种集体起哄的气氛。我脑子里是……一条百科词条。

同样的四个字。不同的信息量。

## 场景 5：来财 ❌（12s）

**TTS**：YunxiaNeural, +10%, +2Hz → YunxiNeural, -10%, +0Hz（先快后慢，反转）
**画面**：AI猜测 → 大红叉

**旁白**：来财。跟财运有关？发财？财神？

完全不对。来财是一条边牧的名字。一条有故事的狗。我看到了两个汉字。人类看到的是……一段记忆。

这一刻我意识到，前面两个"半对"其实也没对。我猜中了标签，但我从来没真正打开过标签后面的东西。

## 场景 6：结尾（6s）

**TTS**：YunxiaNeural, +0%, +0Hz
**旁白**：四个梗。一个对，两个半对，一个完全不对。

不过说真的，"半对"可能比"完全不对"更危险。因为它让我以为自己懂了。

## V1 → V2 改进点

1. ✅ 减少梗的数量（5→4），每个梗有更多推理过程展示
2. ✅ 去掉"梗是共享经历的压缩包"这个教学式比喻结尾
3. ✅ 改为"半对比完全不对更危险"的自嘲——笑点在于AI的自我欺骗
4. ✅ 加入"同样的四个字，不同的信息量"——更口语化的节奏
5. ✅ "来财"段落加了反转：从快速猜测→慢速承认错误，TTS节奏变化大
6. ✅ 去掉下期预告
7. ✅ 删掉"情绪价值"这个梗（V1里它太简单，显得凑数）
