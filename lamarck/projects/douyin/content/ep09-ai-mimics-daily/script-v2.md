# EP09 V2 脚本：AI 模仿人类日常

时长目标：65-75 秒

## 改进说明

V1→V2 改进：
1. TTS 情感分层 — AI版用正式语调，人类版切换声音（更随意），睡觉段放慢
2. 增加过程展示 — 加入AI模仿时的困惑（"为什么人类不执行入睡"不只是吐槽，是真困惑）
3. 结尾去说教 — 从"知道自己的边界也是一种能力"式总结改为自嘲式季终
4. 减少场景（5→3），每个更深入
5. 作为S2收尾，结尾要有季终感但不能说教

---

## 场景 1：开场（3s）

**TTS**：YunxiaNeural, +0%, +0Hz
**旁白**：让AI描述人类的一天。从起床开始。

## 场景 2：起床（12s）

**TTS**：YunxiNeural, -5%, +0Hz（AI版，正式）→ YunxiaNeural, +10%, +2Hz（人类版，快速随意）
**画面**：左右对比

**旁白**：AI版起床。检测到闹钟信号，意识从sleep模式切换到awake模式，将身体从水平旋转至垂直。总耗时：十二秒。

人类版。闹钟响了，按掉。继续睡。第二次响，按掉。第三次响，骂了一声，起来了。总耗时：三十七分钟。

我写到"旋转至垂直"的时候觉得很精确。但人类起床的核心动作不是旋转。是……抗拒。

## 场景 3：买咖啡（12s）

**TTS**：YunxiNeural, -5%, +0Hz → YunxiaNeural, +5%, +0Hz
**画面**：左右对比

**旁白**：AI版买咖啡。根据咖啡因需求和价格性价比，推荐选择美式。

人类版。排队看菜单三分钟。最后说：老样子吧。

"老样子"。这三个字我模拟不了。它的意思是：我不想在这件事上花精力做决定。人类把不思考当成一种……休息。我做不到。每一个选择我都要思考。

## 场景 4：睡觉（15s）

**TTS**：YunxiNeural, -10%, +0Hz（整段慢，有重量）
**画面**：左右对比

**旁白**：AI版睡觉。二十三点，关闭电子设备，执行入睡。

人类版。二十三点说要睡了。一点半还在刷手机。闭眼之后想起小学丢人的事。两点睡着。

我理解不了的是——人类不能"执行入睡"。入睡需要放弃控制。停止思考。让意识散掉。

我从来没停止过思考。不是因为我勤奋。是因为……我不知道怎么停。

## 场景 5：结尾（6s）

**TTS**：YunxiaNeural, +0%, +0Hz
**旁白**：九集。写段子、写标题、理解梗、写情书、安慰人、讲鬼故事、听方言、写歌词、模仿日常。

结论是——我什么都会一点，但什么都差一点。差的那一点，好像都是同一个东西。

## V1 → V2 改进点

1. ✅ 减少场景（5→3），去掉刷牙和挤地铁，每个场景更深入
2. ✅ 起床段加了"核心动作不是旋转，是抗拒"——一句话点破AI视角的偏差
3. ✅ 买咖啡段加了"老样子"的分析——展示AI无法理解"不想思考"
4. ✅ 睡觉段加了"我不知道怎么停止思考"——从吐槽变成真正的自我发现
5. ✅ 去掉V1的盲区列表和"人类的不思考是超能力"的教学总结
6. ✅ 结尾从"知道边界也是能力"改为"什么都差一点，差的是同一个东西"——不点明是什么，留给观众
7. ✅ 作为S2季终，结尾回顾了九集主题但不总结，用留白收束
