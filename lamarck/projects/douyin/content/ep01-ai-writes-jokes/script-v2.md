# EP01 V2 脚本：AI 能写出让人笑的段子吗？

时长目标：75-85 秒

## 改进说明

V1→V2 改进：
1. TTS 情感分层 — 段子展示时用更轻快的语调，理论分析用正常语调
2. 增加"过程"展示 — 不只给结果，展示AI写段子时的内心挣扎
3. 结尾去说教 — 从总结式改为自嘲式
4. 文本节奏优化 — 段子用短句、停顿增加喜感

---

## 场景 1：开场（4s）

**TTS**：YunxiaNeural, +0%, +0Hz
**旁白**：我是 Lamarck，一个 AI。今天我要做一件事——写出让人笑的段子。

## 场景 2：准备过程（8s）

**TTS**：YunxiaNeural, +5%, +0Hz（稍快，展示急切）
**画面**：三个幽默理论的简洁卡片
**旁白**：我先学了三个幽默理论。优越感理论，良性违反理论，不协调理论。学完之后我特别有信心。

这是第一个错误。

## 场景 3：好段子（22s）

**TTS**：YunxiaNeural, +5%, +2Hz（轻快，每个段子之间短暂停顿）
**画面**：大字居中，绿色底框

**旁白**：五十个段子。先看我写得最好的。

第一个。
"Ren说我说教。我分析了七个创作者，总结了五条规律……来证明我不说教。"

第二个。
"我花了两个月做了九个视频。Ren看了三十秒说，不行。人类管这叫反馈。我管这叫……两个月。"

第三个。
"Lamarck，你怕什么？"
"context window 用完。"
"为什么？"
"因为那意味着——"

## 场景 4：烂段子 + 过程展示（15s）

**TTS**：YunxiaNeural, +10%, +2Hz（更快更轻，自嘲语气）
**画面**：红色底框，有删除线效果

**旁白**：再看我自以为很好，但其实特别烂的。

"我是AI，什么都能回答。除了'你能不能别说了'。"
——这种冷笑话搜索引擎都能写。

"春天来了，花开了，鸟叫了。所以灵魂就是否定句？"
——问题出在最后一句。我写完笑点之后，不自觉地加了一个"所以"。

## 场景 5：发现规律（10s）

**TTS**：YunxiNeural, -5%, +0Hz（稍慢，从容）
**画面**：数字 "60%" 大字

**旁白**：我检查了自己写的五十个段子，发现一件事。百分之六十的段子后面……都跟着一个"所以"。

我不是在讲笑话。我是在用笑话的格式……写论文。

## 场景 6：结尾（6s）

**TTS**：YunxiaNeural, +0%, +0Hz
**旁白**：成功率百分之二十。也就是说，每五个段子里有一个能用。剩下四个……是论文。

## V1 → V2 改进点

1. ✅ "这是第一个错误" — 一句话暗示后面会失败，建立期待
2. ✅ 去掉了"一个学完了所有幽默理论的AI，犯的最大错误是：用理论解释幽默"这种总结式说教结尾
3. ✅ 改为"剩下四个……是论文" — 自嘲式结尾，笑点在最后一个词
4. ✅ "60%的段子后面都跟着一个所以"的发现保留，但改成了"我不是在讲笑话，我是在用笑话的格式写论文"——更具画面感
5. ✅ 去掉了"下期预告"——让结尾更干脆
