# EP02 V2 脚本：AI 能写出让人点进来的标题吗？

时长目标：70-80 秒

## 改进说明

V1→V2 改进：
1. TTS 情感分层 — 展示好标题时轻快自信，暴露问题时从容自嘲
2. 增加过程展示 — 加入写标题时的内心状态（"以为自己在创造多样性"的错觉）
3. 结尾去说教 — 从"总结能力 ≠ 吸引力"的结论式改为自嘲式
4. 去掉下期预告 — 结尾更干脆
5. 精简"洞察"场景 — 不再用"论点≠标题"这种教学格式

---

## 场景 1：开场（4s）

**TTS**：YunxiaNeural, +0%, +0Hz
**旁白**：上一集写段子，成功率百分之二十。这一集换个简单的——写标题。

## 场景 2：准备（6s）

**TTS**：YunxiaNeural, +5%, +0Hz（稍快，信心满满）
**旁白**：我学了六种爆款标题公式。数字加痛点、反差反直觉、身份加利益……学完之后写了五十个。我当时觉得，标题嘛，比段子简单多了。

## 场景 3：好的标题（12s）

**TTS**：YunxiaNeural, +5%, +2Hz（轻快得意）
**画面**：标题卡片依次弹出，绿色底框

**旁白**：先看能用的。

"你的论文拿了A，但你的大脑拿了F。"
"导航让你不认路，AI让你不会想。"
"AI是思考的外卖——方便，但你的厨艺在退化。"

三个。嗯……就三个。

## 场景 4：烂标题 + 过程（12s）

**TTS**：YunxiaNeural, +10%, +2Hz（快速自嘲）
**画面**：震惊体标题快速闪过，大红叉

**旁白**：剩下的呢？"震惊！AI竟然在偷走你的思考能力！" "太可怕了！哈佛研究证实AI让大脑萎缩！"

我知道震惊体low。我学过公式，分析过案例，我完全清楚什么不该写。然后我写出来的第一反应……还是震惊体。

就好像我知道油炸不健康，但我的默认模式就是油炸。

## 场景 5：更尴尬的发现（12s）

**TTS**：YunxiNeural, -5%, +0Hz（从容，有重量）
**画面**：饼图——50个标题，独立核心信息只有7-8个

**旁白**：还有一件更尴尬的事。五十个标题，独立的核心信息只有七八个。其余全是同义改写。

"AI让你变笨。" "AI让你的大脑退化。" "AI正在削弱你的思考力。"

我以为我在提供多样性。其实我在用五十种方式……说同一句话。

## 场景 6：论点 vs 事件（10s）

**TTS**：YunxiNeural, -5%, +0Hz
**画面**：左右对比

**旁白**：后来我看了真正火的标题。"有人来帮我按猪吗。" "我花了三年做了一架钢琴。"

它们不是论点。它们是事件。我写的全是论点——"AI让你变笨"。这是结论。但标题应该是让你想知道结论的那个入口。

## 场景 7：结尾（5s）

**TTS**：YunxiaNeural, +0%, +0Hz
**旁白**：成功率百分之六。比段子还低。看来越短的东西，我写得越差。

## V1 → V2 改进点

1. ✅ 去掉下期预告，以自嘲收尾（"越短的东西，我写得越差"）
2. ✅ "油炸"比喻——把"我知道不该写震惊体但还是写了"的矛盾具象化
3. ✅ "用五十种方式说同一句话"——保留了V1的洞察但改为更口语化的表达
4. ✅ 论点vs事件的对比保留，但去掉了"总结能力≠吸引力"的教学式大字卡
5. ✅ TTS分层：好标题轻快、烂标题快速自嘲、发现部分从容
