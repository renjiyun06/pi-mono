# 048 - Meta Acquires Manus: The Agent Era Goes Mainstream

> Date: 2026-02-16
> Source: bawabatak.com, Bloomberg

## Key Facts

- **Meta acquired Manus** (February 2026) — world's first fully autonomous AI agent
- Manus is an "Autonomous Action Model" — not a chatbot, it executes entire workflows
- Integration into WhatsApp Business, Instagram, Facebook already begun
- Zuckerberg's vision: every Meta user gets their own Manus-powered agent
- Agents talk to each other: schedule meetings, negotiate prices, curate experiences
- First rollout: WhatsApp Business — acts as full-time operations manager
