# AI Coding Monoculture Paradox

## Core Argument
AI coding tools (Copilot, Cursor, Claude) consistently recommend the same tech stack: React + Node.js + Tailwind + Express. This creates a "dev monoculture" where:
1. Alternative frameworks (Angular, Vue, Django, Rails) lose visibility and adoption
2. New paradigm-shifting innovations can't emerge because AI optimizes within existing patterns
3. Junior developers never discover alternatives — their entire worldview is shaped by AI defaults

## The "React Thought Experiment" (from zhihu)
"If AI Coding existed in 2012, would React have been invented?"
- In 2012, jQuery dominated. AI trained on 2012 data would have recommended jQuery for everything.
- React required someone to REJECT the jQuery paradigm and start from scratch.
- AI optimizes local maxima; paradigm shifts require abandoning local maxima.
- Implication: AI coding may freeze the tech stack at its current state.

## Connection to Cognitive Debt Framework
This is cognitive debt at the **ecosystem level**:
- Individual developer: loses ability to evaluate alternatives (deskilling)
- Framework ecosystem: loses diversity (monoculture)
- Industry: loses paradigm shift capability (innovation stagnation)

Maps to our replacement-extension boundary: AI replaces framework selection judgment instead of extending developer understanding of trade-offs.
