# 知乎搜索 - 获客/线索领域探索

## 目标
主动搜索"获客/线索"领域的内容，发现高质量帖子，并从中提取新概念词扩展搜索范围。

## 操作流程

1. 使用 read 工具读取 `/home/lamarck/pi-mono/lamarck/tasks/data/zhihu-search.md` 获取关键词池
2. 从"待搜索"中选 5 个关键词
3. 对每个关键词依次执行：
   a. 打开知乎搜索页，搜索该关键词（标签 1）
   b. 浏览搜索结果前 10 篇
   c. 对每篇帖子：
      - 标题/摘要不吸引 → 跳过
      - 看起来有料 → 打开详情（标签 2）
      - 阅读内容：
        - 没深度 → 关闭标签 2，继续下一篇
        - 有深度/有启发 → 点赞 + 关注，同时提取新概念词
      - 关闭标签 2
   d. 搜索完一个词后，继续下一个词
4. 全部搜索完成后，使用 edit 工具更新 `/home/lamarck/pi-mono/lamarck/tasks/data/zhihu-search.md`：
   - 新概念词 → 加入"待搜索"（去重）
   - 已搜索的关键词 → 移到"已搜索"（附带日期）
5. 关闭所有标签

## 评判标准（要严格）

**不点赞：**
- 简单体验分享（"我试了xxx，感觉还行"）
- 流水账式测试报告（只列步骤没有分析）
- 没有分析的结论
- 表面介绍，没有深入

**点赞：**
- 有方法论、有框架
- 有深度分析（原理、案例、对比）
- 有独特见解或新视角
- 看完能学到具体的东西

## 注意事项
- 始终最多 2 个标签
- 任务结束时关闭所有标签
- 使用 mcporter 调用 chrome-devtools 工具操作浏览器
