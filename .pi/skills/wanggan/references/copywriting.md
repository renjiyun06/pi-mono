# 文案改写示例

## 核心改写思路

### 1. 说明书 → 体验式

| 说明书写法 | 网感写法 |
|-----------|---------|
| 本品采用先进清洁技术，有效去除污垢 | 这玩意竟然把我十年的黑头都吸出来了 |
| 产品功效强大 | 用了一次我妈都问我怎么回事 |
| 性价比高 | 这价格不冲是傻子 |
| 操作简单 | 懒人闭眼抄作业 |
| 效果显著 | 室友问我是不是整容了 |
| 使用面积广 | 全家老小都能用 |

### 2. 形容词 → 具体场景

| 抽象形容 | 具体场景 |
|---------|---------|
| 很好吃 | 好吃到我舔盘子 |
| 很便宜 | 比我食堂吃一顿还便宜 |
| 很方便 | 躺床上也能操作 |
| 很有效 | 三天就看到变化 |
| 质量好 | 摔了三次还没坏 |
| 很舒服 | 穿上就不想脱下来 |

### 3. 平淡陈述 → 情绪冲击

| 平淡陈述 | 情绪冲击 |
|---------|---------|
| 今天分享一个好物 | 姐妹们！这个必须让你们知道 |
| 这个产品很好用 | 谁懂啊！用完我原地结婚 |
| 值得推荐 | 你们怎么不早告诉我有这个东西 |
| 我买了一个东西 | 这钱花得我心甘情愿 |
| 效果不错 | 我直接封它为年度最佳 |

## 改写案例

### 案例1：护肤品

**❌ 原版**
> 这款面霜含有玻尿酸成分，具有保湿补水的功效，适合干性肌肤使用。

**✅ 改后**
> 换季脸干到脱皮的姐妹看过来！这个面霜我连用了一周，现在脸滑得跟剥了壳的鸡蛋一样。

**改了什么**：
- 开头直接点痛点"脸干到脱皮"
- 用第一人称体验"我连用了一周"
- 效果用比喻"剥了壳的鸡蛋"而不是"保湿补水"

---

### 案例2：数码产品

**❌ 原版**
> 这款充电宝容量20000毫安，支持快充，体积小巧便于携带。

**✅ 改后**
> 出差三天我就带了这一个，手机电脑全靠它续命。关键才巴掌大，塞包里完全没感觉。

**改了什么**：
- 用场景"出差三天"代替参数"20000毫安"
- "续命"比"充电"更有情绪
- "巴掌大"比"体积小巧"更具体

---

### 案例3：食品

**❌ 原版**
> 本店招牌菜，选用新鲜食材，口感鲜美，欢迎品尝。

**✅ 改后**
> 这道菜我们家卖了十年了，老客户专门从隔壁区开车来吃。不好吃你直接骂我。

**改了什么**：
- 用"卖了十年"建立信任
- "从隔壁区开车来"比"欢迎品尝"更有说服力
- "不好吃你骂我"制造反差，显得自信

---

## 万能改写公式

### 公式1：痛点 + 解决 + 效果
> [目标人群的痛点] → [用了这个之后] → [具体效果/变化]

### 公式2：场景 + 体验 + 结论
> [什么情况下用] → [用的过程/感受] → [一句话总结]

### 公式3：对比 + 惊喜
> [以前怎样/试过什么] → [用了这个之后的反差]
